# FinanceGPT

FinanceGPT is a domain-specific Large Language Model (LLM) fine-tuned for Indian stock market and finance-related question answering.

## Tech Stack

- Python
- Hugging Face Transformers
- Microsoft Phi-3 Mini
- QLoRA
- PEFT
- TRL
- Gradio
- Google Colab
- Hugging Face Spaces
